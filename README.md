# 📊 Retail Sales Analysis Dashboard

## Overview

This project analyzes retail sales data using Python for data preprocessing and Power BI for interactive dashboard visualization.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Power BI
- Jupyter Notebook

## Workflow

1. Load retail sales dataset
2. Clean and preprocess data using Pandas
3. Perform exploratory data analysis
4. Export cleaned dataset
5. Build interactive Power BI dashboard

## Dashboard KPIs

- Total Sales
- Total Profit
- Total Quantity
- Sales by Category
- Sales by Region
- Profit by Category
- Sales Trend by Year
- Top Customers
- Sales by Segment
